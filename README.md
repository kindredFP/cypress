Cypress Proof of Concept to hit Rest APIs

1.  npm install
2.  npm run cypress:open or npm run e2e

cypress:open
-  opens cypress in UI mode

cypress e2e 
-  run in headless mode 
-  uses cypress run --spec to run a specific file

Note:

To run it locally:
npm run cypress:open
